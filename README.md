🧩 Sudoku Suite – Генератор и решатель Судоку на 7 языках
Мощный инструмент для создания и решения головоломок Судоку любого уровня сложности.
Генерируй уникальные доски, решай любые Судоку (включая неполные), проверяй решение и анализируй сложность.
Реализован на 7 языках программирования с единым интерфейсом командной строки и расширенными возможностями.

🚀 Возможности
Генерация доски – создание валидного Судоку с уникальным решением (алгоритм заполнения + удаление ячеек).

Решение – решает любую доску 9×9 с помощью оптимизированного бэктрекинга с выбором наилучшей ячейки (Minimum Remaining Values).

Уровни сложности – лёгкий (easy), средний (medium), сложный (hard) – регулируется количеством удалённых ячеек.

Проверка – проверяет, является ли доска корректным Судоку (валидные строки, столбцы, блоки).

Подсчёт решений – определяет количество возможных решений (для проверки уникальности).

Экспорт/импорт – ввод/вывод доски в формате строки из 81 символа (0 для пустых ячеек) или в формате 9×9.

Красивый вывод – цветная сетка с разделителями блоков 3×3.

Анимация решения – пошаговое решение с задержкой (опция --animate).

Статистика – время решения и количество шагов бэктрекинга.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [режим] [опции]
Режимы
Режим	Описание
generate	Сгенерировать новое Судоку (доступны опции -l для уровня)
solve	Решить Судоку из файла или строки
check	Проверить корректность доски
count	Подсчитать количество решений
export	Экспортировать доску в строку (81 символ)
Опции
Опция	Описание
-i, --input <file|string>	Входная доска (файл или строка 81 символа)
-o, --output <file>	Сохранить результат в файл
-l, --level <easy|medium|hard>	Уровень сложности (для генерации)
-a, --animate	Анимация решения (задержка между шагами)
-q, --quiet	Только вывод доски (без дополнительной информации)
-h, --help	Справка
Примеры
bash
# Сгенерировать лёгкое Судоку и вывести
python sudoku.py generate -l easy

# Решить Судоку из файла
python sudoku.py solve -i puzzle.txt -o solution.txt

# Проверить корректность
python sudoku.py check -i "530070000600195000098000060800060003400003001700020006060000280000419005000080079"

# Подсчитать количество решений (с анимацией)
python sudoku.py count -i puzzle.txt -a

# Экспортировать доску в строку
python sudoku.py export -i puzzle.txt

🧠 Архитектура и алгоритмы
Генерация:

Заполнение диагональных блоков 3×3 случайными числами.
Решение с помощью бэктрекинга для получения заполненной доски.
Удаление ячеек в зависимости от уровня сложности (easy: 30–35, medium: 40–45, hard: 50–55).
Проверка уникальности решения (если больше одного — удаление отменяется).
Решение:

Оптимизированный бэктрекинг с выбором пустой ячейки с минимальным количеством кандидатов (MRV).

Анимация решения с регулируемой задержкой.

Проверка:

Валидация всех строк, столбцов и блоков 3×3.

Проверка на дубликаты и корректность значений (1–9).

Подсчёт решений:

Модифицированный бэктрекинг с подсчётом всех возможных решений (ограничение для предотвращения бесконечности).

✨ Дополнительные фичи
Цветной вывод – выделение строк, столбцов, блоков.

Работа с файлами – чтение доски из текстового файла (9 строк по 9 цифр, 0 для пустых).

Импорт из строки – компактный формат из 81 символа.

Статистика решения – время, количество шагов, сложность.

Автосохранение – результат автоматически сохраняется в файл при указании -o.
